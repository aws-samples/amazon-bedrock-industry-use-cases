# Mortgage and Lending Document Processing with Amazon Bedrock Data Automation

This project demonstrates how to use Amazon Bedrock Data Automation (BDA) to automate mortgage and lending document processing. The solution extracts structured data from common mortgage documents such as loan applications, income verification, and property appraisals.

## Overview

The mortgage and lending industry requires processing large volumes of complex documents with high accuracy. This project shows how to leverage BDA custom blueprints to extract key fields from mortgage documents, reducing manual review time and improving processing consistency.

## Prerequisites

- AWS account with Bedrock Data Automation access
- Python environment with required dependencies
- S3 bucket for document storage and processing

For BDA fundamentals, start with the [Getting Started notebooks](../../common/bedrock-data-automation-intro/).

## Getting Started

1. Configure AWS credentials
2. Install required Python packages (boto3, pandas)
3. Open `21_mortgage_and_lending.ipynb` and follow the walkthrough
