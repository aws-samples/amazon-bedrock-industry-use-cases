# Getting Started with Bedrock Data Automation (BDA)

This folder contains introductory notebooks for understanding Amazon Bedrock Data Automation. Start here before exploring the industry-specific BDA use cases.

## Notebooks

1. [Getting Started - How BDA works](11_getting_started_with_bda.ipynb)
2. [Document Insights with Standard Outputs](12_standard_output_extended.ipynb)
3. [Custom Document Insights with Blueprints](13_custom_outputs_and_blueprints.ipynb)

## Prerequisites

- AWS account with Bedrock Data Automation access (available in `us-east-1` and `us-west-2`)
- Python environment (JupyterLab in SageMaker Studio or equivalent)
- IAM role with BDA permissions (see [BDA documentation](https://docs.aws.amazon.com/bedrock/latest/userguide/bda-projects.html))
